初识gulp
发表于 2017-11-12| 更新于 2019-11-21|js
阅读量:

node和npm，cnpm安装就不多言了。

全局安装gulp

cnpm install gulp -g

gulp -v 输入版本，则说明安装成功。

新建package.json

{  "name": "sai",  "version": "1.0",  "description": "gulp css!",  "author": {    "name": "13sai",    "email": "95742781@qq.com"
  },  "license": "MIT",  "devDependencies": {    "gulp": "^3.9.1",    "gulp-minify-css": "^1.2.4"
  }
}

本地安装gulp，gulp-minify-css。

cnpm install gulp --save-dev cnpm install gulp-minify-css --save-dev

新建gulpfile.js

var gulp = require('gulp'),
    mincss = require('gulp-minify-css');//定义一个sai任务gulp.task('sai', function () {
    gulp.src('assets/css/*.css') //该任务针对的文件
        .pipe(mincss()) //该任务调用的模块
        .pipe(gulp.dest('assets/mini')); //将会在assets/mini下生成css});
 
gulp.task('default',['sai']); //定义默认任务

gulp

会看到任务sai开始，完成，这样就完成了gulp压缩css。

此外，gulp还可以压缩js，img。
