评论实时刷新效果

2016-04-12 web js 评论 字数统计: 723(字) 阅读时长: 3(分)

话不多说，直接上示例。

准备：前台jquery1.8  后台thinkphp3.1 数据库mysql 表comment（字段c_id, m_nick_name, weibo_id, w_content, insert_time, member_id等）


html:

<ul class="weibo-menu">
  <li>转发</li>
  <li><a class="pinglun" data-rel="24" onclick="getComment(24)">评论</a></li>
  <li>赞&nbsp;/li>
</ul>
<div class="comment-module-24" style="display:none;">
  <input type="text" class="comment-24" placeholder="想说点什么"/>
  <div>
    <button class="am-btn" type="button" onclick="getComment(24)">评论</button>
  </div>
  <div class="am-padding comment-list-24"></div>
</div>

js:

$(function() {
  $('.pinglun').on('click', function() {
          //评论列表的显示隐藏
      id2 = $(this).attr('data-rel');
      $('.comment-module-'+id2).toggle();
  });
  
});


function getComment(id){
    // 获取对应id
    comment = $('.comment-' + id).val();
    // ajax刷新
    $.post("{:U('getComment')}", { w_id: id , w_comment: comment}, function(data){
      //json格式
      data = JSON.parse(data);
      //初始化变量
      html = '';   
      //循环输出评论列表并存入html变量
      $.each(data, function(i, item) {
        html += '<div> '+ item.m_nick_name + '：'+item.w_content + '<br/>' + item.insert_time + '</div><hr/>'
    });
      $(".comment-list-"+id).html(html);
      $('.comment-' + id).val('');
    });
};


后台：

public function getComment(){
    $c = M("comment");
    $weibo_id = I('w_id');
    $w_content = I('w_comment');
    //判断是否有评论传入
    if($w_content == ''){
        $list = $c->where("weibo_id=%d",$weibo_id)->order('c_id desc')->limit(5)->select();
        echo json_encode($list);
    }else{
        $data['weibo_id'] = $weibo_id;
        $data['w_content'] = $w_content;
        $data['member_id'] = session('m_id');
        $r = $c->add($data);
        $list = $c->where("weibo_id=%d",$weibo_id)->order('c_id desc')->limit(6)->select();
        echo json_encode($list);
    }
        
}


PS:

定义和用法

post() 方法通过 HTTP POST 请求从服务器载入数据。

语法

jQuery.post(url,data,success(data, textStatus, jqXHR),dataType)
参数描述
url必需。规定把请求发送到哪个 URL。
data可选。映射或字符串值。规定连同请求发送到服务器的数据。
success(data, textStatus, jqXHR)可选。请求成功时执行的回调函数。
dataType

可选。规定预期的服务器响应的数据类型。

默认执行智能判断（xml、json、script 或 html）。
