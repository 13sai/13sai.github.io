thinkphp之定义控制器
发表于 2016-02-21| 更新于 2019-11-21|php
阅读量:

thinkphp3.2版为例，

控制器通常需要继承系统的Controller类或者其子类，例如，定义一个\Home\Controller\IndexController控制器类： 



<span class="pun">&lt;?</span><span class="pln">php</span>

<span class="kwd">namespace</span><span class="typ">Home</span><span class="pln">\Controller</span><span class="pun">;</span>

<span class="kwd">use</span><span class="typ">Think</span><span class="pln">\Controller</span><span class="pun">;</span>

<span class="kwd">class</span><span class="typ">IndexController</span><span class="kwd">extends</span><span class="typ">Controller</span><span class="pun">{</span>

<span class="kwd">public</span><span class="kwd">function</span><span class="pln"> hello</span><span class="pun">(){</span>

<span class="pln"> echo </span><span class="str">hello</span><span class="pun">;</span>

<span class="pun">}</span>

<span class="kwd">public</span><span class="kwd">function</span><span class="pln"> test</span><span class="pun">(){</span>

<span class="pln"> echo </span><span class="str">test</span><span class="pun">;</span>

<span class="pun">}</span>

<span class="pun">}</span>

控制器的名称采用驼峰法命名（首字母大写），控制器文件位于Home/Controller/IndexController.class.php。

IndexController控制器类的hello和test方法就是操作方法，访问下面的URL地址： 



<span class="pln">http</span><span class="pun">:</span><span class="com">//serverName/Home/Index/hello</span>

<span class="pln">http</span><span class="pun">:</span><span class="com">//serverName/Home/Index/test</span>

会分别输出： 



<span class="pln">hello</span>

<span class="pln">test</span>

操作方法的定义必须是公共方法，否则会报操作错误，所以，下面的操作定义只能访问hello操作，而不能访问test操作。 



<span class="pun">&lt;?</span><span class="pln">php</span>

<span class="kwd">namespace</span><span class="typ">Home</span><span class="pln">\Controller</span><span class="pun">;</span>

<span class="kwd">use</span><span class="typ">Think</span><span class="pln">\Controller</span><span class="pun">;</span>

<span class="kwd">class</span><span class="typ">IndexController</span><span class="kwd">extends</span><span class="typ">Controller</span><span class="pun">{</span>

<span class="kwd">public</span><span class="kwd">function</span><span class="pln"> hello</span><span class="pun">(){</span>

<span class="pln"> echo </span><span class="str">hello</span><span class="pun">;</span>

<span class="pun">}</span>

<span class="kwd">protected</span><span class="kwd">function</span><span class="pln"> test</span><span class="pun">(){</span>

<span class="pln"> echo </span><span class="str">test</span><span class="pun">;</span>

<span class="pun">}</span>

<span class="pun">}</span>

因为操作方法就是控制器的一个方法，所以遇到有和系统的关键字冲突的方法可能就不能定义了，这个时候我们可以设置操作方法的后缀来解决，例如： 



<span class="str">ACTION_SUFFIX</span><span class="pun">=&gt;</span><span class="str">Action</span><span class="pun">,</span><span class="com">// 操作方法后缀</span>

设置操作方法的后缀为Action，这样，控制器的操作方法定义调整为： 



<span class="pun">&lt;?</span><span class="pln">php</span>

<span class="kwd">namespace</span><span class="typ">Home</span><span class="pln">\Controller</span><span class="pun">;</span>

<span class="kwd">use</span><span class="typ">Think</span><span class="pln">\Controller</span><span class="pun">;</span>

<span class="kwd">class</span><span class="typ">IndexController</span><span class="kwd">extends</span><span class="typ">Controller</span><span class="pun">{</span>

<span class="kwd">public</span><span class="kwd">function</span><span class="pln"> listAction</span><span class="pun">(){</span>

<span class="pln"> echo </span><span class="str">list</span><span class="pun">;</span>

<span class="pun">}</span>

<span class="kwd">public</span><span class="kwd">function</span><span class="pln"> helloAction</span><span class="pun">(){</span>

<span class="pln"> echo </span><span class="str">hello</span><span class="pun">;</span>

<span class="pun">}</span>

<span class="kwd">public</span><span class="kwd">function</span><span class="pln"> testAction</span><span class="pun">(){</span>

<span class="pln"> echo </span><span class="str">test</span><span class="pun">;</span>

<span class="pun">}</span>

<span class="pun">}</span>


操作方法的后缀设置只是影响控制器类的定义，对URL访问没有影响。
