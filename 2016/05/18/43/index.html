去除a链接的虚线框
发表于 2016-05-18| 更新于 2019-11-27|css

a链接上的虚线框非常常见，而且真的非常难看。如图，

如何去除呢？

只需要加一行代码，完美解决。

a{
blr:expression(this.onFocus=this.blur()); /*for IE */ 
outline:none; /*for FF */ 
}


谷歌系还有一个输入框高亮显示的智能模式，其实也不怎么好看，去除加入：

input,textarea{outline: none;}
