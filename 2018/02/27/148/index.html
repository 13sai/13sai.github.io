近期用到的若干Linux命令
发表于 2018-02-27| 更新于 2019-11-21|Linux
阅读量:

查看并发数

netstat -n | awk '/^tcp/ {++S[$NF]} END {for(a in S) print a, S[a]}'

返回结果示例： LAST_ACK 5   (正在等待处理的请求数) 
SYN_RECV 30 ESTABLISHED 1597 (正常数据传输状态) 
FIN_WAIT1 51 FIN_WAIT2 504 TIME_WAIT 1057 (处理完毕，等待超时结束的请求数) 
 
状态：描述 
CLOSED：无连接是活动的或正在进行 
LISTEN：服务器在等待进入呼叫 
SYN_RECV：一个连接请求已经到达，等待确认 
SYN_SENT：应用已经开始，打开一个连接 
ESTABLISHED：正常数据传输状态 
FIN_WAIT1：应用说它已经完成 
FIN_WAIT2：另一边已同意释放 
ITMED_WAIT：等待所有分组死掉 
CLOSING：两边同时尝试关闭 
TIME_WAIT：另一边已初始化一个释放 
LAST_ACK：等待所有分组死掉

修改文件时间

touch -d "6:03pm 05/06/2000" file

find /data/logs/ -mtime +30 -name '*' -exec rm -rf {} ;

find /data/logs/  -name "*[1,2]" -print;

find /data/logs/ -mtime +20 -name "*[1]" -exec rm -rf {} ;

find /data/logs/ -mtime +40 -name '*[0-9].log' -exec rm -rf {} ;

# 删除过期日志0 4 * * * www find /data/logs/weixin -mtime +32 -name '*[0-9].log' -exec rm -rf {} \;
