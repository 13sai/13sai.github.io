LNMP环境搭建
发表于 2018-07-29| 更新于 2019-11-21|os
阅读量:

最近公司有新项目上线，新买了服务器，便需要搭建环境，便实操了一下。


  1. lnmp包装


lnmp的安装时比较轻松的，一行代码，

wget http://soft.vpser.net/lnmp/lnmp1.5.tar.gz -cO lnmp1.5.tar.gz && tar zxf lnmp1.5.tar.gz && cd lnmp1.5 && ./install.sh lnmp

然后会提示你选择的nginx版本，输入mysql密码，php版本，选择之后坐等安装，大概需要半个小时。


如果显示Nginx: OK，MySQL: OK，PHP: OK。

就表示安装成功了。


2. 扩展安装（以redis扩展为例）

wget https://codeload.github.com/phpredis/phpredis/zip/develop -O phpredis.zip
unzip phpredis.zip
cd /phpredis-develop
/usr/local/php/bin/phpize./configure --with-php-config=/usr/local/php/bin/php-config
make && make install   //编译安装

在php.ini中加入

extension=redis.so

重启php-fpm，通过php -m命令查看。
